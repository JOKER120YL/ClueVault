# ClueVault v2 Fast Archive Demo

这是 ClueVault v2 的一次性静态流程样机，用于验证“从微信收到用户模型后，快速归档到共享盘对应专业文件夹”的主流程。

## 使用方式

直接用浏览器打开：

```text
index.html
```

## 验证范围

- 悬浮窗点击入口。
- 悬浮窗模拟拖入文件。
- 页面内桌面置顶悬浮窗预览。
- 悬浮窗展开 / 收起形态。
- 收起态只显示卡通角色。
- 达到当日归档阈值后自动从蛙弟切换到蜂哥。
- 拖入时预留张开大口状态。
- 独立快速归档小面板。
- 蛙弟 / 蜂哥形象切换预览。
- 可选开启拖入后截图提示。
- 模拟框选聊天截图，并一并归档。
- 专业文件夹按钮点选。
- 项目名、用户来源、备注默认折叠在更多信息中。
- 归档成功结果。
- 共享盘不可写失败状态。
- 今日归档记录。
- 今日归档记录修改备注，并模拟同步 `归档信息.md`。
- 独立经验库入口和经验草稿保存。

## 不验证

- 真实文件复制。
- 真实共享盘写入。
- 真实 Windows 悬浮窗能力。
- AI 总结。
- 截图 OCR。
- 团队经验库共享。
- 最终技术架构。
