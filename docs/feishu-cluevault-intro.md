# ClueVault 软件介绍与使用说明

## 1. 这是什么

ClueVault 是一个 Windows 桌面模型归档助手。它的目标很简单：运营或客服在微信群收到用户模型、压缩包、截图后，可以快速把文件归档到共享盘对应专业目录，方便测试组后续取用和验证。

当前版本优先解决“快速归档文件”这件事，不要求提交人写完整问题描述，也不要求先整理成正式问题单。

（配图：ClueVault 使用流程）

## 2. 适用场景

- 微信群里收到用户发来的模型文件、压缩包或截图。
- 需要把材料交给测试组，但暂时还没有完整的问题结论。
- 运营每天处理大量用户反馈，希望少填字段、少切窗口、快速完成归档。
- 后续需要查看“今天归档了哪些文件”。

## 3. 下载与运行

当前发布包是免安装压缩包。

使用方式：

1. 下载 `ClueVault-v0.2.0-win-x64.zip`。
2. 解压整个文件夹。
3. 双击 `ClueVault.Desktop.exe` 启动。
4. 不要只单独复制 exe 文件运行，程序需要同目录下的配套文件。

## 4. 首次配置

首次打开后，需要先完成基础配置：

- `姓名/昵称`：用于写入归档记录，方便测试组知道是谁提交的。
- `共享盘目录`：选择测试组可访问的公共盘根目录。
- `蜂哥切换阈值`：用于悬浮窗状态展示，归档越多，蜂哥状态会逐步变化。

配置完成后，桌面会出现悬浮窗。以后日常归档主要通过悬浮窗完成。

## 5. 日常归档流程

1. 在微信、资源管理器或桌面中找到用户发来的模型文件。
2. 把文件拖到桌面悬浮窗上。
3. 弹出快速归档面板后，选择专业：建筑、结构、暖通、给排水、电气或其他。
4. 点击 `归档到共享盘`。
5. 归档完成后，可以在主窗口的 `今日记录` 查看结果。

归档时默认来源为微信群。项目名、备注等信息默认不强制填写，避免增加日常操作负担。

## 6. 归档结果

软件会把文件复制到共享盘对应专业目录，并生成一份 `归档信息.md`。

示例结构：

```text
共享盘目录/
  建筑/
    2026-05-28_微信群_张三_153012/
      用户模型.rvt
      归档信息.md
```

`归档信息.md` 会记录提交人、来源、专业、归档时间、文件清单和备注，方便后续追溯。

## 7. 桌面快捷方式

如果希望更方便启动，有两种方式：

- 在软件的 `配置中心` 点击 `创建桌面快捷方式`。
- 或在解压目录中双击 `CreateDesktopShortcut.bat`。

创建后，桌面会出现 `ClueVault` 快捷方式。

## 8. 主窗口、悬浮窗与退出

- 关闭主窗口不会直接退出软件，软件会隐藏到右下角托盘。
- 双击右下角托盘图标，可以重新打开主窗口。
- 右键托盘图标，可以选择打开主窗口或退出软件。
- 右键桌面悬浮窗，也可以选择打开主窗口或退出软件。

## 9. 更新软件

软件内置了版本信息入口：

1. 打开主窗口。
2. 进入 `关于` 页面。
3. 查看当前版本号。
4. 点击 `检查更新`，打开 GitHub Releases 页面查看新版本。

## 10. 常见问题

### 看不到主窗口怎么办？

查看 Windows 右下角托盘区域，双击 ClueVault 图标即可重新打开主窗口。

### 如何彻底退出软件？

右键托盘图标，选择退出。也可以右键桌面悬浮窗，选择退出。

### 归档失败怎么办？

优先检查共享盘目录是否存在、当前电脑是否有写入权限，以及文件是否正在被其他软件占用。

### 归档专业选错了怎么办？

当前版本建议手动到共享盘中调整目录。后续版本可以继续补充“修改专业并同步移动文件”的功能。

### 经验库功能现在能用吗？

当前版本先聚焦模型归档。经验库、截图自动总结、导入导出等功能会作为后续能力继续深化，不影响现阶段归档使用。

## 11. 当前边界

- 当前版本不接入真实 AI。
- 当前版本不强制填写问题描述。
- 当前版本主要用于共享盘归档，不替代完整测试管理系统。
- 当前发布包是免安装版，不是安装向导版。
