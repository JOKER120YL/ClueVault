# ClueVault v0.2.0 发布说明

## 下载与运行

本版本为 Windows x64 自包含版本，解压后运行 `ClueVault.Desktop.exe`。

首次打开需要配置：

- 姓名 / 昵称
- 共享盘目录
- 蜂哥切换阈值

## 主要功能

- 桌面悬浮窗接收文件拖入。
- 快速归档到共享盘专业目录。
- 自动生成 `归档信息.md`。
- 今日归档记录。
- 主窗口关闭后隐藏到系统托盘。
- 托盘菜单和悬浮窗右键菜单支持打开主窗口和退出。
- 关于页面包含版本、项目地址、更新入口和更新日志。
- 支持在配置中心创建桌面快捷方式。
- 发布包内提供 `CreateDesktopShortcut.bat`，可一键创建桌面快捷方式。

## 已知限制

- 当前为免安装版，不写入开始菜单和桌面快捷方式。
- 检查更新会打开 GitHub Releases 页面，尚未实现自动下载和替换。
- 经验库为入口展示，完整导入、导出和 AI 总结后续实现。
