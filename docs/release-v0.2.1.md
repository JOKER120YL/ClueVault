# ClueVault v0.2.1 发布说明

## 推荐下载

新用户优先下载并运行：

- `ClueVault-v0.2.1-setup-win-x64.exe`

安装包会安装到当前用户目录，不需要管理员权限，并可创建桌面快捷方式和开始菜单入口。

## 用于软件内更新

软件内自动更新会下载：

- `ClueVault-v0.2.1-win-x64.zip`

这个 zip 仍然需要随 Release 一起上传。后续版本也要保持相同命名规则：

```text
ClueVault-v版本号-win-x64.zip
```

## 主要变化

- 新增基础自动更新机制。
- 关于页面的“检查更新”会检查 GitHub 最新 Release。
- 发现新版本后，可自动下载 zip、退出当前程序、覆盖安装目录并重启。
- 新增 Inno Setup 安装包，方便同事安装、创建桌面快捷方式和开始菜单入口。
- 安装目录默认在当前用户目录，减少管理员权限和更新覆盖问题。

## 注意事项

- 已安装 `v0.2.0` 的用户需要手动升级到 `v0.2.1` 一次，因为 `v0.2.0` 本身没有自动更新逻辑。
- 从 `v0.2.1` 开始，后续版本可以通过软件内“检查更新”完成更新。
- 如果公司网络限制 GitHub 下载，软件会提示失败，用户仍可打开 GitHub Releases 手动下载。
