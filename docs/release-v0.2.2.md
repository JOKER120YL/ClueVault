# ClueVault v0.2.2 发布说明

## 推荐下载

新用户优先下载并运行：

- `ClueVault-v0.2.2-setup-win-x64.exe`

## 修复内容

- 修复安装器创建桌面快捷方式时写入 `C:\Users\Public\Desktop` 导致权限不足的问题。
- 安装器现在改为给当前用户创建桌面快捷方式，不需要管理员权限。

## 仍需上传的自动更新包

软件内自动更新会下载：

- `ClueVault-v0.2.2-win-x64.zip`

发布 GitHub Release 时请同时上传：

```text
ClueVault-v0.2.2-setup-win-x64.exe
ClueVault-v0.2.2-win-x64.zip
ClueVault-v0.2.2-win-x64.zip.sha256
```
