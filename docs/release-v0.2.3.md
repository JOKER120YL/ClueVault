# ClueVault v0.2.3 发布说明

## 推荐下载

新用户优先下载并运行：

- `ClueVault-v0.2.3-setup-win-x64.exe`

## 主要变化

- 修复“今日记录”跨天仍显示昨天归档的问题。
- 今日记录增加 `今日归档 / 历史归档` 切换。
- 主面板支持点击 `粘贴截图`，从剪贴板导入截图并随模型一起归档。
- 主面板在快速归档页支持 `Ctrl + V` 粘贴剪贴板截图。
- 悬浮窗快速归档面板增加 `粘贴剪贴板截图` 按钮。
- 增加本地日志文件，便于排查异常和未响应问题。

## 日志位置

```text
%LOCALAPPDATA%\ClueVaultDesktop\cluevault.log
```
