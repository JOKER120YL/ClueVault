# ClueVault v0.2.5 发布说明

## 推荐下载

本版本修复 `v0.2.4` 覆盖安装后首次启动可能出现的空引用异常。建议下载并运行：

- `ClueVault-v0.2.5-setup-win-x64.exe`

## 主要变化

- 修复主窗口初始化时，`今日归档 / 历史归档` 筛选按钮提前触发事件导致的启动异常。
- 保留 `v0.2.4` 中对自动更新器的修复：失败时保留窗口并写入 `update.log`。

## 异常原因

`v0.2.4` 中新增的记录筛选按钮在 XAML 加载阶段会触发 `Checked` 事件。此时另一个筛选控件还没有完成初始化，代码访问该控件时触发空引用。

本版本已增加启动期空值保护，窗口加载完成后仍会正常应用默认的今日归档筛选。

## 文件说明

- `ClueVault-v0.2.5-setup-win-x64.exe`：首次安装或覆盖安装使用。
- `ClueVault-v0.2.5-win-x64.zip`：软件内自动更新使用。
- `ClueVault-v0.2.5-win-x64.zip.sha256`：校验更新包完整性。

普通用户只需要下载并运行 `.exe` 安装包。
