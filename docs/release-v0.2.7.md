# ClueVault v0.2.7

本版本主要优化多人同时使用时的归档稳定性，并让共享盘上的归档文件夹更容易识别。

## 更新内容

- 归档文件夹命名改为“日期_来源_归档人_项目名或模型名”，不再默认追加秒数。
- 未填写项目名时，自动使用第一个模型文件名作为文件夹后缀。
- 如果确实出现同名归档目录，才自动追加 `-2`、`-3` 这类短序号，避免覆盖。
- 团队统计写入改为按客户端拆分统计文件，降低多人同时写入共享盘时的冲突风险。
- 统计看板兼容旧版统计文件，升级后仍可读取历史记录。
