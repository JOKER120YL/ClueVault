# 2026-05-28 v2 MVP 开发记录

## 已完成

- 新增 `docs/architecture-decision-v2.md`，明确正式 MVP 走 `WPF / .NET 8` 主线。
- 新增 `docs/mvp-implementation-plan-v2.md`，拆分归档服务、快速面板、透明悬浮窗、今日记录、截图辅助、经验库阶段。
- `PRD v2` 的技术架构章节从“待决”更新为“已决策”。
- WPF 主线新增 `ArchiveService` 语义，归档输出改为 `归档信息.md`。
- `QuickSubmitWindow` 已改为 `QuickArchiveWindow`，命名和功能语义对齐。
- 悬浮窗拖入文件后改为弹出独立快速归档面板，不再先唤起主窗口。
- 快速归档面板默认只显示文件和专业，项目名、来源、备注折叠到“更多信息”。
- WPF 悬浮窗去掉圆形底、文字条和计数徽章，窗口层保持透明。
- 正式悬浮窗素材已接入 WPF：默认 `frogdii-idle.png`，达到阈值后按数量切换 `ebee-stern.png`、`ebee-angry-overload.png`、`ebee-breakdown.png`。
- 悬浮窗跳动动画已移除，避免常驻桌面时干扰工作。
- 拖入文件后的快速归档面板改为手动定位，优先显示在悬浮窗上方。
- 配置完成后启动程序不再默认弹出主窗口，只显示悬浮窗；主窗口通过双击悬浮窗打开。
- 设置页新增版本号和“检查更新”入口，当前版本 `v0.2.0`。
- 新增系统托盘图标：托盘右键可打开主窗口、隐藏主窗口、退出程序；双击托盘图标打开主窗口。
- 主窗口点击关闭按钮时改为隐藏，避免后台仍在运行但用户找不到入口。
- 悬浮窗右键菜单新增“打开主窗口”和“退出 ClueVault”。
- “检查更新”按钮已连接到 GitHub Releases：`https://github.com/JOKER120YL/ClueVault/releases`。
- WPF 主窗口已重做为 v2 信息架构：左侧菜单、快速归档、今日记录、经验库、配置中心。
- 主窗口旧“问题提交”视觉已替换，归档主路径突出为“拖文件 / 选专业 / 归档到共享盘”。
- 新增软件图标：`Assets/app-icon.ico` 用于 exe、窗口标题栏和系统托盘，预览图保存在 `assets/design/cluevault-app-icon-1024.png`。
- 软件图标已简化为小尺寸任务栏优先版本：只保留绿色底和蛙弟头部识别，不再包含平板、文件角标或头盔文字。
- 左下角状态卡片文案已从内部阈值表达改为“今日已归档 / 当前形象”，降低理解成本。
- 本机 `%LOCALAPPDATA%\ClueVaultDesktop` 已移动到备份目录，用于模拟新安装初始状态。
- 图标进一步改为剪影式任务栏图标，参考 Clash Verge 这类小尺寸识别方式，不承载文字和插画细节。
- 主窗口右上角“设置”入口已移除，设置能力统一收敛到左侧“配置中心”。
- 配置中心新增当前配置摘要：使用者、共享盘目录、蜂哥切换阈值、更新说明。
- 首页按钮层级调整为“归档到共享盘”为主按钮，“选择文件”为次按钮。
- 新增“关于”页面：集中展示版本信息、项目地址、GitHub Releases 更新入口和更新日志。
- 设置窗口已精简为配置项，不再展示版本更新和日志。
- 任务栏图标改为纯净蛙头图标，透明背景，避免小尺寸下信息过载。
- 已生成 Windows x64 自包含发布包：`dist/ClueVault-v0.2.0-win-x64.zip`。
- 发布包支持桌面快捷方式：配置中心可创建快捷方式，发布目录内也包含 `CreateDesktopShortcut.bat`。
- 发布包 SHA256：`657505D6A086BFA66818C84DFA916C76323A7ABFE2E0B7BAFC505322987FD5CA`。

## 注意事项

- 旧版蛙弟 / 蜂哥 PNG 素材本身不是透明底，代码层透明后仍可能看到图片自带矩形底；新接入的 4 张半身素材已验证为透明 PNG。
- 旧版 `frogdi-confused-final.png` 和 `ebee-angry-final.png` 暂时保留，后续确认无依赖后再清理。
- 主窗口仍有部分旧“问题提交”文案，后续统一清理。
- 截图辅助和经验库还未进入正式 WPF 实现。
- 完整自动更新尚未实现下载 / 替换 / 重启流程；当前先打开 GitHub Releases 页面。
